HTML Reference

Embedded Content Tags

Tag Name Description
<img> Image The <img> tag allows you to add an image to a web page. <img> tag is a self-closing tag.
<picture> Picture
<source> Source
<iframe> Iframe
<video> Video The <video> element requires a src attribute with a link to the video source. Unlike the <img> element however, the <video> element requires an opening and a closing tag.
<audio> Audio The <audio> element is used to embed audio content into a document. Like <video>, <audio> uses src to link the audio source.
<track> Track
